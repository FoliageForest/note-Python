### type()

```python
print('使用print直接输出字面量的类型信息')

print(type('黑马程序员'))
print(type(666))
print(type(114.514))

print('用变量存储type()的返回值')

string_type = type('黑马程序员')
int_type = type(666)
float_type = type(114.514)

print(string_type)
print(int_type)
print(float_type)

print('查看变量中存储的数据类型')

name = '黑马程序员'
name_string = type(name)
print(name_string)

print('name_type如何存储变量的类型信息')
print(type(name_string))
```

### P021 演示字符串的三种定义方法，字符串包含引号的写法

```python
# 单引号定义法
name = 'heimachengxuyuan'
print(type(name))
# 双引号定义法
name = "heimachengxuyuan"
print(type(name))
# 三引号定义法，写法和多行注释是一样的
name = """heima
chengxu
yuan
"""
print(type(name))

name = '"heima"chengxuyuan"'
print(name)
name = "'heima'chengxuyuan'"
print(name)
# 使用转义字符
name = 'hei\'ma\"chen\'xu\'yuan\"'
print(name)
```
